# Practice6

Контрольные вопросы:
1.Какие основные типы памяти используются в OpenCL?
Global, Local, Constant, Private.
2.Как настроить глобальную и локальную рабочую группу?
Через параметры clEnqueueNDRangeKernel: global_work_size — общее количество work-items, local_work_size — размер одной рабочей группы (должно делить global без остатка).
3.Чем отличается OpenCL от CUDA?
OpenCL — открытый кросс-платформенный стандарт (NVIDIA, AMD, Intel и др.), CUDA — проприетарная технология только для NVIDIA с лучшей производительностью и инструментами.
4.Какие преимущества дает использование OpenCL?
Кросс-платформенность, переносимость кода между разными производителями GPU, открытый стандарт, независимость от одного вендора.
